Privacy Policy

Last updated: March 2026

IMDB Click Race: Six Degrees of Kevin Bacon ("the Extension") is a multiplayer browser game. This policy explains what data is collected and how it is used.

Data We Collect

The Extension collects the following minimal data to operate the multiplayer game:

Display name — a name you choose yourself, used to identify you in the game lobby and leaderboard. This is never linked to any real identity.

Anonymous session ID — a randomly generated identifier created by Firebase Authentication each time you use the Extension. It is not linked to your Google account or any personal information.

Game activity — click counts, finish times, and the path of actors you clicked during a round. This data is used to run the game and determine winners.

How Data Is Stored

Game data is stored in Google Firebase Realtime Database. Old game sessions are automatically cleaned up. No data is stored on our own servers.

Data Sharing

We do not sell, share, or transfer your data to any third party. Game data is only visible to players in the same game lobby.

Third-Party Services

The Extension uses the following third-party services:

Firebase (Google) — for real-time game state and anonymous authentication. Firebase's privacy policy is available at firebase.google.com/support/privacy.

Oracle of Bacon — to calculate the optimal connection path between two actors at the end of each round. No personal data is sent to this service.

IMDB — the Extension operates on IMDB pages but does not collect or transmit any data from those pages.

Children's Privacy

The Extension does not knowingly collect data from children under 13.

Contact

If you have any questions about this privacy policy, please open an issue on the extension's GitHub repository.
